EquivariantModels.CategoricalBasis — Type

CategoricalBasis : defines the discrete 1p basis

\[ \phi_q(u) = \delta{u, U_q},\]

where $U_q, q = 1, \dots, Q$ are finitely many possible values that the variable $u$ may take. Suppose, e.g., we allow the values [:a, :b, :c], then 

P = CategoricalBasis([:a, :b, :c]; varsym = :u, idxsym = :q)
evaluate(P, State(u = :a))     # Bool[1, 0, 0]
evaluate(P, State(u = :b))     # Bool[0, 1, 0]
evaluate(P, State(u = :c))     # Bool[0, 0, 1]

If we evaluate it with an unknown state we get an error: 

evaluate(P, State(u = :x))   
# Error : val = x not found in this list

Warning : the list of categories is internally stored as an SVector which means that lookup scales linearly with the number of categories
